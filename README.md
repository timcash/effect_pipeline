### Effect Pipeline
For simple testable code

### Setup instructions
Make sure you have nvm installed
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.1/install.sh | bash
```

Install the project
```
git clone git@github.com:timcash/effect_pipeline.git
cd effect_pipeline
nvm use
npm install
npm test
```
